Github+Hexo创建个人博客

步骤

git part

1）安装git 下载地址

2）打开git bash，设置配置信息

git config --global user.name "github用户名"
git config --global user.email "github注册邮箱"

3)设置密匙，省去每次输入密码的步骤
a.git生成密匙（私钥），复制生成的id_rsa.pub内容 

ssh-keygen -t rsa -C "github注册邮箱"

b.设置github密匙（公钥）
打开github_settings_keys，输入id_rsa.pub内容
c.验证公钥设置是否成功

ssh git@github.com

Node.js

1)安装Node.js 下载地址

2)检测安装是否成功

node -v
npm -v

Hexo

1)创建blog文件夹

2)安装Hexo

npm install -g hexo-cli

3)初始化博客，即在文件夹下创建博客

hexo init blog

4)进入博客，快乐玩耍

#创建博客
hexo new md
#生成网页
hexo g
#启动服务预览，即可在本地预览网页
hexo s

5)打开浏览器地址localhost:4000，即可进入网站进行预览

6)推送网站
a.关联Hexo与Github，修改站点配置文件

deploy:
    type: git
    repo: 仓库地址
    branch: master

即配置网站部署地址

b.安装hexo-git部署插件

c.输入指令

#清除缓存
hexo clean
#生成网页
hexo g
hexo d

7)常用的Hexo命令

npm install hexo -g
npm update hexo -g
hexo init
hexo n
hexo g
hexo s
hexo d
#用于网页不正常时
hexo clean

8)更换主题

ROS安装

步骤

参考文章
1）添加镜像源以及配置密匙 

sudo sh -c '. /etc/lsb-release && echo "deb http://mirrors.ustc.edu.cn/ros/ubuntu/ `lsb_release -cs` main" > /etc/apt/sources.list.d/ros-latest.list'  
sudo apt-key adv --keyserver 'hkp://keyserver.ubuntu.com:80' --recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654   
sudo apt-get update

2)安装ROS
sudo apt install ros-melodic-desktop-full

3)配置ROS的全局环境变量

echo "source /opt/ros/melodic/setup.bash" >> ~/.bashrc  
source ~/.bashrc

4)安装打包工具依赖 

sudo apt install python-rosdep python-rosinstall    python-rosinstall-generator python-wstool build-essential

5)初始化rosdep 

sudo rosdep init   
rosdep update

rosdep可以轻松地为要编译的代码添加系统依赖项，并且帮助我们运行ROS中核心组件

报错

参考文章

解决方法：更改系统软件下载设置与软件下载源同步
参考文章1
参考文章2
rosdep update超时

ROS安装Gazebo、Griz等关键组件

安装Gazebo,连接Gazebo与ROS

参考文章
1)添加软件源 

sudo sh -c 'echo "deb http://packages.osrfoundation.org/gazebo/ubuntu-stable `lsb_release -cs` main" > /etc/apt/sources.list.d/gazebo-stable.list
wget https://packages.osrfoundation.org/gazebo.key -O - | sudo apt-key add -

2)安装Gazebo 

sudo apt-get update
sudo apt-get install gazebo9
sudo apt-get install libgazebo9-dev

3)安装ROS-Gazebo接口

sudo apt install ros-melodic-gazebo-ros-pkgs  ros-melodic-gazebo-msgs  ros-melodic-gazebo-plugins  ros-melodic-gazebo-ros-control

4)验证是否连接成功

roscore
rosrun gazebo_ros gazebo
