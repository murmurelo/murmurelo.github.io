部署环境

Apache

1)安装Apache服务及其扩展包

yum -y install httpd httpd-manual mod_ssl mod_perl mod_auth_mysql

2)启动Apache服务

systemctl start httpd.service

3)安全组

4)打开浏览器输入服务器的公网IP，得到测试页面表明Apache服务安装成功

MySQL

1）在服务器上搭建MySQL数据库

#还不太理解这个命令的含义
wget http://dev.mysql.com/get/mysql57-community-release-el7-10.noarch.rpm
#下载相关包
yum -y install mysql57-community-release-el7-10.noarch.rpm
yum -y install mysql-community-server --nogpgcheck

2）启动MySQL数据库

systemctl start mysqld.service
#查看运行状态
systemctl status mysqld.service
#查看初始密码
grep "password" /var/log/mysqld.log
#登录数据库
mysql -uroot -p
#修改初始密码
ALTER USER 'root'@'localhost' IDENTIFIED BY 'NewPassWord1.';
#在MySQL下创建一个存储博客网站内容
create database wordpress; 
#查看是否创建成功
show databases;
#退出数据库
exit

PHP

1)安装PHP

yum -y install php php-mysql gd php-gd gd-devel php-xml php-common php-mbstring php-ldap php-pear php-xmlrpc php-imap

2)创建PHP测试页面

echo "<?php phpinfo(); ?>" > /var/www/html/phpinfo.php

3)重启Apache服务（可能是因为要刷新网站）

systemctl restart httpd

4)访问网址http://<ECS公网IP>/phpinfo.php

WordPress

1)安装WordPress

yum -y install wordpress

2)修改WordPress配置文件

# 进入/usr/share/wordpress目录。
cd /usr/share/wordpress
# 修改路径。
ln -snf /etc/wordpress/wp-config.php wp-config.php
# 查看修改后的目录结构。
ll
# 在Apache的根目录/var/www/html下，创建一个wp-blog文件夹
mkdir /var/www/html/wp-blog
#移动wordpress到Apache目录
mv * /var/www/html/wp-blog/
#修改wp-config.php配置文件
sed -i 's/database_name_here/wordpress/' /var/www/html/wp-blog/wp-config.php
sed -i 's/username_here/root/' /var/www/html/wp-blog/wp-config.php
sed -i 's/password_here/NewPassWord1./' /var/www/html/wp-blog/wp-config.php
#查看配置文件信息是否修改成功
cat -n /var/www/html/wp-blog/wp-config.php

3）重启Apache服务

systemctl restart httpd

Github+Hexo创建个人博客

参考文章

步骤

git part

1）安装git 下载地址

2）打开git bash，设置配置信息

git config --global user.name "github用户名"
git config --global user.email "github注册邮箱"

3)设置密匙，省去每次输入密码的步骤
a.git生成密匙（私钥），复制生成的id_rsa.pub内容 

ssh-keygen -t rsa -C "github注册邮箱"

b.设置github密匙（公钥）
打开github_settings_keys，输入id_rsa.pub内容

c.验证公钥设置是否成功

ssh git@github.com

Node.js

1)安装Node.js 下载地址

2)检测安装是否成功

node -v
npm -v

Hexo

1)创建blog文件夹

2)安装Hexo

npm install -g hexo-cli

3)初始化博客，即在文件夹下创建博客

hexo init blog

4)进入博客，快乐玩耍

#创建博客
hexo new md
#生成网页
hexo g
#启动服务预览，即可在本地预览网页
hexo s

5)打开浏览器地址localhost:4000，即可进入网站进行预览

6)推送网站
a.关联Hexo与Github，修改站点配置文件

deploy:
    type: git
    repo: 仓库地址
    branch: master

即配置网站部署地址

b.安装hexo-git部署插件

c.输入指令

#清除缓存
hexo clean
#生成网页
hexo g
hexo d

7)常用的Hexo命令

npm install hexo -g
npm update hexo -g
hexo init
hexo n
hexo g
hexo s
hexo d
#用于网页不正常时
hexo clean

步骤

参考文章

sudo sh -c '. /etc/lsb-release && echo "deb http://mirrors.ustc.edu.cn/ros/ubuntu/ `lsb_release -cs` main" > /etc/apt/sources.list.d/ros-latest.list'  
sudo apt-key adv --keyserver 'hkp://keyserver.ubuntu.com:80' --recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654   
sudo apt-get update

2)安装ROS
sudo apt install ros-melodic-desktop-full

3)配置ROS的全局环境变量

echo "source /opt/ros/melodic/setup.bash" >> ~/.bashrc  
source ~/.bashrc

4)安装打包工具依赖 

sudo apt install python-rosdep python-rosinstall    python-rosinstall-generator python-wstool build-essential

5)初始化rosdep 

sudo rosdep init   
rosdep update

rosdep可以轻松地为要编译的代码添加系统依赖项，并且帮助我们运行ROS中核心组件
