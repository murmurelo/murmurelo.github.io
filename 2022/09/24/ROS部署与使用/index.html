ROS安装

步骤

参考文章
1）添加镜像源以及配置密匙 

sudo sh -c '. /etc/lsb-release && echo "deb http://mirrors.ustc.edu.cn/ros/ubuntu/ `lsb_release -cs` main" > /etc/apt/sources.list.d/ros-latest.list'  
sudo apt-key adv --keyserver 'hkp://keyserver.ubuntu.com:80' --recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654   
sudo apt-get update

2)安装ROS
sudo apt install ros-melodic-desktop-full

3)配置ROS的全局环境变量

echo "source /opt/ros/melodic/setup.bash" >> ~/.bashrc  
source ~/.bashrc

4)安装打包工具依赖 

sudo apt install python-rosdep python-rosinstall    python-rosinstall-generator python-wstool build-essential

5)初始化rosdep 

sudo rosdep init   
rosdep update

rosdep可以轻松地为要编译的代码添加系统依赖项，并且帮助我们运行ROS中核心组件

参考文章
Ubuntu20.04下需将melodic改为noetic,依赖包安装发生变化，且不再需要对rosdep进行初始化

ROS安装Gazebo、Griz等关键组件

安装Gazebo,连接Gazebo与ROS

参考文章
1)添加软件源 

sudo sh -c 'echo "deb http://packages.osrfoundation.org/gazebo/ubuntu-stable `lsb_release -cs` main" > /etc/apt/sources.list.d/gazebo-stable.list
wget https://packages.osrfoundation.org/gazebo.key -O - | sudo apt-key add -

2)安装Gazebo 

sudo apt-get update
sudo apt-get install gazebo9
sudo apt-get install libgazebo9-dev

3)安装ROS-Gazebo接口

sudo apt install ros-melodic-gazebo-ros-pkgs  ros-melodic-gazebo-msgs  ros-melodic-gazebo-plugins  ros-melodic-gazebo-ros-control

4)验证是否连接成功

roscore
rosrun gazebo_ros gazebo

参考文章
